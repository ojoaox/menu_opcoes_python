# 🧠 Terminal MultiTool em Python

Um programa interativo de terminal desenvolvido em Python que reúne várias funcionalidades úteis em um único menu, com interface estilizada usando **Rich** e suporte a **voz** com `pyttsx3`.

---

## 🚀 Funcionalidades

O sistema possui um menu principal com as seguintes opções:

### 🔢 1. Calculadora

* Operações básicas: soma, subtração, multiplicação e divisão
* Tratamento de erros (ex: divisão por zero)
* Loop contínuo até o usuário decidir sair

---

### 📅 2. Data e Hora

* Exibe a **data atual**
* Exibe a **hora atual**
* Leitura em voz das informações

---

### 📺 3. Lista de Animes

Gerenciamento completo de uma lista:

* ➕ Adicionar anime
* ➖ Remover anime por índice
* 📋 Visualizar lista formatada
* 🔢 Ver quantidade total
* 🔤 Ordenar lista

---

### 💬 4. Mensagem Motivacional

* Exibe uma mensagem aleatória
* Leitura em voz para o usuário

---

### 🧹 9. Limpar Tela

* Limpa o terminal (Windows)

---

### ❌ 0. Sair

* Encerra o programa com mensagem de despedida

---

## 🛠️ Tecnologias Utilizadas

* Python 3
* [Rich](https://pypi.org/project/rich/) → Interface no terminal
* [pyttsx3](https://pypi.org/project/pyttsx3/) → Síntese de voz
* Bibliotecas nativas:

  * `math`
  * `datetime`
  * `os`
  * `random`

---

## 📦 Instalação

Clone o repositório ou copie o arquivo `.py`:

```bash
git clone https://github.com/ojoaox/menu_opcoes_python.git
cd menu_opcoes_python
```

Instale as dependências:

```bash
pip install rich
pip install pyttsx3
```

---

## ▶️ Como Executar

```bash
python main.py
```
